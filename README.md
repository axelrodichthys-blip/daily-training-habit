# 毎日トレーニング習慣

決めた時間（寝る前10分など）に、少しでもストレッチ・筋トレを続けるための習慣化アプリです。

- 📱 スマホ向け PWA（ホーム画面に追加してアプリのように使えます）
- ⏱ メニューを選ぶと、種目ごとにタイマー付きでガイド再生
- 🔥 連続日数（ストリーク）・通算・カレンダーで記録
- 💪 おまかせメニュー4種（寝る前ストレッチ / 朝の筋トレ / 肩こり首ほぐし / 腰お尻ストレッチ）
- 🔒 記録はお使いの端末の中だけに保存（外部に送信されません）

「1種目でもやれば達成」として記録される設計です。**少しでもOK。続けることが目標。**

## 使い方（スマホ）

1. 公開URLをスマホのブラウザで開く
2. 「ホーム画面に追加」を選ぶ
3. 追加されたアイコンから起動

## 確実なリマインドについて

Webアプリはアプリを閉じている間の「時刻ぴったり通知」が苦手です（特に iPhone）。
確実に毎日リマインドしたい場合は、**端末の時計アプリのアラームやリマインダーで同じ時刻に1つ通知をセット**し、鳴ったらこのアプリを開くのがおすすめです。
