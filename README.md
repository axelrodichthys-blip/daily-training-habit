# ナイトレ🌙

寝る前に、少しでもストレッチを続けるための習慣化アプリです。

- 📱 スマホ向け PWA（ホーム画面に追加してアプリのように使えます）
- ⏱ 寝る前ストレッチ5種目（約15分）を、種目ごとにタイマー付きでガイド再生
- 🎵 ストレッチ中にBGM（デフォルト2曲／スマホから最大10曲アップロード・並び替え可）
- 👑 通算の達成日数（7日／30日／50日ごと）でリワード達成バッジ。カレンダーに残り、タップでお祝い表示
- 🔥 連続日数（ストリーク）・通算・カレンダーで記録
- 🔒 記録・曲はお使いの端末の中だけに保存（外部に送信されません）

「1種目でもやれば達成」として記録される設計です。**少しでもOK。続けることが目標。**

## 使い方（スマホ）

1. 公開URLをスマホのブラウザで開く
2. 「ホーム画面に追加」を選ぶ
3. 追加されたアイコンから起動

## 確実なリマインドについて

Webアプリはアプリを閉じている間の「時刻ぴったり通知」が苦手です（特に iPhone）。
確実に毎日リマインドしたい場合は、**端末の時計アプリのアラームやリマインダーで同じ時刻に1つ通知をセット**し、鳴ったらこのアプリを開くのがおすすめです。
